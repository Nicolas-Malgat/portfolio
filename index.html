Le projet Immothep

"La société Immothep, nouveau fleuron de l'immobilier Français, souhaite créer un module d'estimation de biens basé sur l'intelligence artificielle pour enrichir sa plateforme et acquérir de nouveaux acheteurs/vendeurs."

Lors de ce projet nous avons travaillé avec des notebooks, nous avons filtré puis entrainé un modèle de régression linéaire à estimer la valeur d'un appartement et d'une maison. Le modèle peut être appelé par une API via un serveur local.

Le projet SmartTweet

"Vous êtes contactés par une entreprise qui a besoin d’évaluer les premières réactions liées à l’annonce de leur produit. Leur principal canal de communication est twitter. Choisissez votre produit et le hashtag qui lui ai associé"

Ce projet a nécessité l'utilisation de l'API twitter et l'analyse de sentiment de Azure via une autre API. Le tout est enregistré dans une base de donnée et affiché sur une page HTML utilisant vueJS. Notre choix a été d'afficher les différents pourcentages d'avis positifs, négatifs et neutre à l'aide d'un diagramme circulaire.

Le code de ce projet est disponible ici.
